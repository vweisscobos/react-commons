### Inputs

All the available inputs are related in the complete example below

```jsx
import React from 'react';
import {
    DateInput,
    NumberInput,
    SelectInput,
    TextInput,
    TimeInput
} from 'react-commons';

class Form extends React.Components {
    render() {
        return (
            <form>
                //  name
                //  birthday
                //  number of fingers
                //  race
                //  what's time is it?
            </form>
        );
    }
}
```
